# 🎵 بوت كلمات الأغاني والأناشيد - Words Songs Bot

بوت تلجرام متخصص في البحث عن كلمات الأغاني والأناشيد، مع نظام اشتراك موحد متكامل مع بوتات الأدوات الاجتماعية الأخرى.

---

## 🌟 **المميزات**

| الميزة | الوصف |
|--------|-------|
| 🔍 **بحث متقدم** | البحث عن الأغاني بالاسم أو جزء من الكلمات |
| 🎲 **اقتراح عشوائي** | عرض أغنية عشوائية من قاعدة البيانات |
| 📄 **تصدير الملفات** | إرسال الكلمات كملف نصي منظم |
| 📊 **إحصائيات** | عرض إحصائيات قاعدة البيانات |
| 💰 **نظام اشتراك** | مجاني 5 بحث/يوم - مميز غير محدود |
| 🔗 **تكامل كامل** | نظام اشتراك موحد مع باقي البوتات |

---

## 📋 **الأوامر المتاحة**

| الأمر | الوصف |
|-------|-------|
| `/start` | بدء الاستخدام |
| `/help` | عرض المساعدة |
| `/about` | معلومات عن البوت |
| `/stats` | إحصائيات قاعدة البيانات |
| `/premium` | معلومات الاشتراك المميز |
| `/random` | اقتراح أغنية عشوائية |

## 💰 **نظام الاشتراك**

### **الخطة المجانية**
- ✅ 5 بحث يومياً
- ✅ جميع المميزات الأساسية

### **الخطة المميزة (10 دولار مدى الحياة)**
- ✅ بحث غير محدود
- ✅ دعم أولوية في المعالجة
- ✅ تحديثات حصرية أولاً
- ✅ يشمل جميع البوتات الخمسة
